# Delichat - Backend

The backend of the real-time chat app **Delichat**, built using **Express.js**, **MongoDB**, and **Socket.IO**. It handles JWT user authentication, message storage, and real-time communication between users.

---

## Technology
- Node.js
- Express.js
- MongoDB
- Mongoose
- Socket.IO
- dotenv
- JWT Authentication
